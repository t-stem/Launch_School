# Description:
- TicTacToe is a game
- It has two players
- It takes place on a 3x3 grid
- The game starts with an empty grid
- The players take turns
- During a turn, a player gets to mark a square with their own marker
- The first player who marks 3 squares in a row (horizontally, vertically or diagonally) wins
*In this game there will be one human and one computer player* ADDED
*In the initial version, the human will get the first turn* ADDED
